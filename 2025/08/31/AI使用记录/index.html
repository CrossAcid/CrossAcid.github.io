AI使用记录

发表于2025-08-31|更新于2025-08-31|学习
|浏览量:

概述

记录一些最近使用的AI和其特点，参考自全网最最全的AI推荐！历时半年，我们选出了Top100【全网首发】_哔哩哔哩_bilibili

通用LLM

LLM 特点/优点 缺点
ChatGPT 泛用性强，有免费套餐无限制使用，输出也很不错 注册需要国外手机号
Claude 编程能力很强 注册需要国外手机号
DeepSeek 使用方便，输出效果略弱于ChatGPT 支持功能较少，且服务器可能会搞人心态
Gemini 泛用性强，整合了谷歌生态，输出也还不错 需要魔法
Grok 泛用性强，整合了X生态，输出也还不错，限制小 需要魔法，据说可能抽风
Kimi 使用方便、各方面的功能都还可以 各方面也就是还可以
豆包 使用方便、各方面的功能都还可以、生态好 各方面也就是还可以（我认为略强于Kimi）
通义千问 使用方便、泛用性强，生态很好，国内前三水准 无显著缺点
元宝 使用方便、支持deepseek、功能比deepseek多点、接入微信生态 无显著缺点
智谱轻言 使用方便、偏数学、理工科科研 无显著缺点

ChatGPT

网址：ChatGPT

需要注册openai账号，需要国外手机号，体验下来算是第一梯队的，问答、生图、深度思考、联网搜索、上传文件都支持

现在无法选择4o、o4-mini、4.1mini之类的了，由后台控制，付费版可以选择模型，默认是4.1mini，无限制使用

有免费套餐，可以满足日常需要

Claude

网址：Claude

也需要国外手机号，比较麻烦，之前实习用Claude 4来写代码体验很好，应该说代码能力很强

主要适用于开发者，代码工作者

不过其AI伦理看的很重，比较多限制

DeepSeek

网址：DeepSeek | 深度求索

国内的开源LLM，体验下来也还不错，但整体而言体验还是比GPT还是差一点

主要是文本生成，功能相对较少，主要是方便 + 免费，可以深度思考+联网搜索，支持文件上传

偶尔会出现服务器响应问题

Gemini

网址：Google Gemini

谷歌出品，使用需要魔法、没有怎么使用过，但评价也是属于第一梯队的

支持深度思考、生图、文档生成、文件生成、上传文件等

有免费套餐，可以选择通用类的2.5 Flash模型，也可以选择偏科研代码之类的2.5 Pro

整合了谷歌生态，算是特点了，之前用它搜索相关研究领域的论文，可能是因为整合了谷歌学术，所以没有给我报假论文，算是可以的

Grok

网址：Grok

马斯克整出来的东西，有限的免费额度，用起来还算方便，谷歌账号就可以登陆使用

支持的功能很多，我甚至看到了定时任务的选项，但没有使用过

试了代码能力和论文检索能力，初步使用起来也还可以。

据说…限制很小~

Kimi

网址：Kimi - 更强大的 AI 助手

之前用的效果不是很好，但是有挺多人推荐，可能是领域不同 + 之前还刚起步的原因

写这个的时候又试了一下（提问：如果要你完成一个cv领域的根据图像检测图像中物体到摄像头之间距离的的python应用，你的思路是什么，给个例子）

输出速度较慢，效果还行（但依然不如其他LLM的回答）

豆包

网址：豆包 - 字节跳动旗下 AI 智能助手

字节的产品，我之前用过，整体效果还是可以的，支持的功能也很多，生态也不错。

属于是综合性很好，日常使用挺不错，但涉及到深层次的东西，可能差点意思

通义千问

网址：通义 - 你的实用AI助手

国内的第一梯队大模型，之前使用下来体验也还可以，但之前代码能力没那么强，但近期应该是有做优化，没有逐个尝试

优势在于国内使用方便，功能也比deepseek多的多，生态很好，在他的网站上也看到智能体了，真的很全面

对于论文这些的检索能力（苦逼研究生）更偏向于国内的论文

元宝

网址：腾讯元宝 - 轻松工作 多点生活

腾讯出品的，支持混元和DeepSeek，因为是腾讯的服务器，所以不容易崩，算是DeepSeek的腾讯版，体验下来也还不错，最近一直用的这个

功能相对较少，但对我够用

它的特点是接入了腾讯生态，他的联网搜索，能检索到微信公众号的文章，这方面是它的优势也是它的劣势，有时候会更专注于微信公众号的结果（但可能那个文章的质量不高）

智谱清言

网址：智谱清言

清华和智谱AI一起做出来，专注于数学、理工科领域，泛用性差一些

稍微体验了下，让它完成一个项目的思路以及代码，体验下来效果还可以，主要是偏向于科研领域

AI生图

这方面我不太关注，但记录一下，之前用过阿里云百炼里那个图像画面扩展

  • DALL-E

网址：DALL·E 3 | OpenAI

老牌多模态产品，基础能力极强，新版文字嵌入出色，支持多轮修改。

  • 商汤秒画

网址：商汤秒画SenseMirage – AI作画，有手就行！还是免费的！

海报生成和二次元风格表现突出，自带提示词不全功能且参数调节自由

  • LibLibAI

网址：LiblibAI-哩布哩布AI - 中国领先的AI创作平台

支持在线生产，可自定义模型，最重要的是社区UGC生态超强

  • 即梦

网址：即梦AI - 一站式AI创作平台

擅长中文字体设计，可用于自媒体封面和海报。兼容复杂画风，内置提示词补全功能。这个我倒是用过，感觉还行

  • 可灵

网址：可灵 AI - 新一代 AI 创意生产力平台

支持复杂提示词的适配。在光影、色彩层次和人物面部细节上表现突出。

视频生成

同生图，不太关注

  • 即梦

网址：即梦AI - 一站式AI创作平台

擅长中文字体设计，可用于自媒体封面和海报。兼容复杂画风，内置提示词补全功能

  • 可灵

网址：可灵 AI - 新一代 AI 创意生产力平台

支持复杂提示词的适配。在光影、色彩层次和人物面部细节上表现突出。

  • 通义万象

网址：通义万相 | 领先的AI视频与图像生成模型

大幅度运动模拟、多对象生成、空间关系理解等维度表现突出，单镜头动态表现稳定。

  • Vidu

网址：Vidu AI - 全球领先的AI内容生产平台

生成视频时角色、场景一致性强，人物细节表现好，复杂镜头的动作和逻辑表现自然，支持首尾帧生视频和电影级运镜

  • Runway

网址：Welcome to Runway - Runway

支持4k视频生成，擅长电影级特效和复杂场景构建。提供视频风格转换、修复等多种编辑工具

  • 海螺

网址：海螺视频：每个想法都是一部大片 – AI视频生成 | 海螺 AI

能力和Runway接近，主体参考功能角色还原度高，适合需要二创的用户

  • Pika

网址：Pika

支持音视频同步的嘴型匹配，适合配音和角色动画。

AI Coding

一些AI IDE

CodeBuddy

网址：Tencent Cloud Code Assistant CodeBuddy – AI Code Editor

腾讯出品的，之前下载体验过，实测效果不错，主要是会生成一堆文件有点烦，但效果可以，目前应该还在逐步内测

此处有个在腾讯实习用的vscode的codebuddy插件（Tencent Cloud CodeBuddy）当时用的内网版很好用，现在这个公开的只能用deepseek-v3，应该差不少

Cursor

网址：Cursor - The AI Code Editor

目前最主流的AI编程工具，基于VSCode开发，Tab体验极佳

GitHub Copilot

集成在vscode里，效果还行，但是额度太少，开发不够用

TRAE

网址：TRAE - The Real AI Engineer

前身是字节的Marscode，虽然只有chat页面，但是功能完整

通义灵码

网址：通义灵码_你的智能编码助手-阿里云

AI功能完整，上手比cursor简单

Agent

AI Agent相关，使用AI帮你完成一个完整的任务，目前仍然在发展中…

AutoGLM

网址：AutoGLM

智谱轻言系列出品，暂未使用

DeepResearch

网址：ChatGPT

ChatGPT的功能之一，具备科研级实力，复杂任务的处理能力很强，但性价比低

Coze

网址：coze.cn

字节旗下的AI智能体平台，上手简单，具备丰富的插件和知识库生态，之前体验过，感觉很有意思

Dify

网址：dify.ai/zh

和Coze类似，但更加自由，上手难度也跟高

Manus

网址：Manus: 一个连接思想和行动的通用 AI 代理

之前非常爆火的一个Agent，能力强大，国内版还未上线

AI 搜索

使用AI帮你搜索一些你想要的东西，减少获取信息所花的时间

秘塔

网址：秘塔AI搜索

时效性强，推理能力强，覆盖面广，最终回答结果非常完善，推荐使用

夸克

网址：夸克官网_电脑版下载_你的AI搜索

夸克的AI搜索，论文相关搜索能力也很强，和秘塔相同水平

perplexity

网址：Perplexity

国外的AI搜索头部应用，但测评据说中文搜索能力和英文差距挺大，英语好可以使用

其他

学习

  • Research Rabbit

网址：ResearchRabbit: AI Tool for Smarter, Faster Literature Reviews

能梳理论文与作者关系、列文献、生成关系网，但论文收录不够全

  • Gauth

网址：Gauth - Best AI Homework Helper for All School Subjects

专注数理解题的产品，支持上传或拍照搜题，可以给出完整的解题过程，支持识别手绘图形，实测可以胜任考研数学

  • Google Learn About

网址：Learn About

支持以知识点的方式记录对话中的关键信息，便于构建知识结构。讲解会搭配视频，提供交互式练习题。

  • NotebookLM

网址：NotebookLM

可帮助用户理解文档中的复杂信息，并提取见解、激发创意。允许从网页链接、YouTube视频链接中整理笔记

  • 多面鹅

网址：OfferGoose 鹅来面(原多面鹅)官网 - AI 面试模拟 | 实时面试提醒 | 通过面试
